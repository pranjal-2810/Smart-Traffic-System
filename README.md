## 📖 Overview

Smart Traffic System is a traffic simulation project designed to demonstrate how intelligent signal control can improve urban traffic management.

The application monitors traffic conditions, dynamically manages traffic signals, schedules signal timing, and provides analytics for efficient traffic movement.

## ✨ Features

- 🚥 Intelligent traffic signal management
- 📊 Traffic analytics and monitoring
- ⏱ Dynamic signal scheduling
- 🚗 Vehicle queue management
- 📈 Real-time traffic simulation

## ⚙ Installation

### Clone the repository

```bash
git clone https://github.com/pranjal-2810/Smart-Traffic-System.git
```

### Navigate into the project

```bash
cd Smart-Traffic-System
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the project

```bash
python main.py
```

## 🧠 System Workflow

1. Initialize traffic simulation.
2. Detect traffic density.
